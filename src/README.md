# Student Result Management System

## Description
A menu-driven Java application to manage student records using Object-Oriented Programming principles.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- Search by Roll Number
- Search by Name
- Sort by Percentage
- Display Rank List
- Display Topper
- Class Statistics
- File Handling
- Exception Handling

## Technologies Used
- Java
- Collections Framework
- File Handling
- OOP

## How to Run

1. Open the project in IntelliJ IDEA.
2. Run `Main.java`.
3. Follow the on-screen menu.
